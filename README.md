## 说明
如有兴趣使用，可使用全部插件或提取部分插件使用.

均不包含插件配置、界面配置信息（个人的WTF文件），需要自行配置各个插件（通常是位置，如zperl）。

部分插件支持命令，例如/bt4，/dbm，等等。其他插件可以在界面-插件里直接进行修正。
